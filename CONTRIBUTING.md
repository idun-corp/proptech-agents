# Contributing to Proptech Agents

Thanks for your interest in contributing!

We welcome contributions from proptech professionals, AI developers, and partners who want to improve or extend the prompt collections.

---

## How to contribute

1. **Fork** this repository.  
2. **Create a branch** for your contribution.  
   ```bash
   git checkout -b add-new-expert-agent
3. **Add or edit prompts** under /experts or /embodied.
- Keep prompt files clear and self-contained.
- Include brief comments explaining intent, context, or safety guidance.
4. ** Commit with a descriptive message:**
   ```bash
   git commit -m "Add energy optimization expert agent prompt"
5. **Open a Pull Request** describing your changes.

### Guidelines

Keep prompts clear, safe, and purposeful.

Avoid including private or confidential data.

Attribution is handled under the CC BY 4.0 license.

All contributions are assumed to be made under the same license.
